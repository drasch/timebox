# timebox

I built this to put behind me while on camera.

## timer

demo: <https://timebox.pages.dev/timebox.html>

keys:

* `r` = reset
* `1` = add 1 minute
* `5` = add 5 minutes
* `8` = add 10 minutes
* `9` = add 25 minutes
* `0` = add 30 minutes
